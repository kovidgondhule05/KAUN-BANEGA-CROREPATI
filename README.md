---

This program is a **console-based quiz game inspired by** *Kaun Banega Crorepati*. It simulates the experience of answering multiple-choice questions to win increasing prize money.

---

### 🔹 Game Introduction

* The program starts by displaying a **welcome message**.
* It explains the basic rule:

  * Choose the correct option to earn more money as you progress.

---

### 🔹 Question Bank

* A list of **15 questions** is stored in the program.

* Each question contains:

  * The question text
  * Four options (A, B, C, D)
  * The correct answer

* The questions cover different topics like:

  * History
  * Science
  * Geography
  * Polity

---

### 🔹 Prize Money Levels

* A separate list stores the **money for each question level**.
* The prize amount increases as the player moves forward in the game.

---

### 🔹 Random Order

* The questions are **shuffled randomly** at the start.
* This ensures the game feels different each time it is played.

---

### 🔹 Game Flow

* The game runs through **15 questions in sequence**.
* For each question:

  * The question and all four options are displayed
  * The player enters their answer (A, B, C, or D)

---

### 🔹 Answer Checking

* The program compares the player’s answer with the correct one:

  * If correct → the player proceeds to the next question and earns more money
  * If incorrect → the game stops immediately

---

### 🔹 Winnings

* After every correct answer:

  * The total prize money is updated
* When the game ends:

  * The final amount won by the player is displayed

---

### 🔹 Game End

* The game ends when:

  * The player answers incorrectly
  * Or completes all questions
* The final winnings are printed at the end

---

### 🔹 Overall Description

This program:

* Simulates a **quiz-based game show**
* Uses **lists to store questions and prize values**
* Uses **randomization** for variety
* Uses a loop to control game progression
* Tracks the player’s progress and winnings
